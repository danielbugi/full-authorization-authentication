# Authentication & Authorization API

Full CRUD MongoDB, Nodejs, authentication and authorization API server,
get access to protected routes with jwt token.
Restricted endpoints that only admin has access.

### Routes:

- Create user
- Login user

#### Restricted routes -

- Get all users
- Get user
- Delete user
- Update user

### Tech used:

- Nodejs
- Express
- MongoDB
- bcrypt
- jsonwebtoken
- mongoose
- morgan
- validator
